# Project Description

This project is a solution for the "Develop a custom evaluation metric" problem from the "Advanced Generative AI" course. It aims to demonstrate the implementation and usage of a custom evaluation metric for generative AI models.

## Course Description

The "Advanced Generative AI" course focuses on advanced concepts and techniques in generative AI, covering topics such as model architectures, training strategies, and evaluation methodologies. This project is part of the coursework, providing practical experience in developing and applying custom evaluation metrics.
